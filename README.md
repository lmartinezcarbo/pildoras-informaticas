# Contact Manager

A desktop CRUD application for managing contacts using Python, Tkinter, and SQLite.

## Features

- Add new contacts
- View all contacts
- Edit existing contacts
- Delete contacts
- Store data locally with SQLite

## Technologies

- Python 3
- Tkinter
- SQLite3
